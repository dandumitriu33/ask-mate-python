Information

Welcome to the Ask Mate questions and answers board.

This is an educational product for Codecool students and the information on this page is fictional. Any resemblance to user data or PII is unintended. All the content on this website is for educational and testing purposes only.

The goal of the exercise is to simulate a Stack Overflow or Quora board with questions, answers, comments and other additional features one might find on these websites.

Night mode

One of the main considerations when the UI choices were made, was the dark colors and white text. This "night mode" version is considered by many to be easier on the eyes. As at this point no JavaScript is used, a toggle is not in place and only "night mode" is available.

Also, "day mode", which resembles the "pen on paper/typewriter" medium, was most likely introduced to make users feel more familiar with the new environment (when computers were first adopted by companies/people).

Usually, heavy computer users prefer a dark background with light color text because the strain on the eyes is reduced during long sessions.

Of course this information is fictional and not backed up by any scientific research but we can't wait for Spreadsheets and Docs to get "night-mode".

Mobile first

This is a principle in web design where websites are designed with mobile devices in mind first and then desktops as mobile use around the world has gone up significantly.

Here is an article from Google about mobile-first indexing: Google Developers

Try it on your phone or if you're on a desktop:
FireFox: Menu > Web Developer > Responsive Design Mode
Chrome: Menu > More tools > Developer tools > top right corner of the tools has a "Toggle device toolbar" button that looks like a phone (or press Ctrl + Shift + M).
