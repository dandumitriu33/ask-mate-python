Information

Welcome to the Ask Mate questions and answers board.

This is an educational product for Codecool students and the information on this page is fictional. Any resemblance to user data or PII is purely fictional and for testing purposes.

The purpose of the exercise is to simulate a Stack Overflow or Quora board with questions, answers, comments and different features one might find on these websites.

Night mode

One of the main considerations for the UI choices is the dark colors and white text. This is easier on the eyes. As at this point no JavaScript is used, a toggle is not in place and only night mode is available.

Also, "day mode", which resembles the "pen on paper/typewriter" medium was most likely introduced to make users feel more familiar with the new environment (when computers were introduced).

Heavy computer users prefer a dark background with light color text because of the strain on the eyes is lower during long sessions.

Of course this information is fictional and not backed up by any scientific research but I can't wait for Spreadsheets to get night-mode.

Mobile first

This is a new principle in web design where websites are designed with mobile devices in mind first and then desktops as mobile use around the world has gone up significantly.

Here is an article from Google about mobile-first indexing: Google Developers

Try it on your phone or if you're on a desktop:
FireFox: Menu > Web Developer > Responsive Design Mode
Chrome: Menu > More tools > Developer tools > top right corner of the tools has a "Toggle device toolbar" button that looks like a phone (or press Ctrl + Shift + M).
